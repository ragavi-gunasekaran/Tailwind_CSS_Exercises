# Visit Sweden - Tailwind CSS Practice

## Overview

This project is a Tailwind CSS practice website based on a **Visit Sweden** theme. It contains 20 exercises to practice different Tailwind CSS utility classes and responsive layouts.

## Features

- Typography and Spacing
- Alert Boxes
- Cards
- Buttons
- Responsive Layouts
- Product Grid
- Hero Section
- Navigation Bar
- Pricing Cards
- Gallery
- Forms and Validation
- Dark Mode
- Dashboard Layout

## Technologies Used

- HTML5
- Tailwind CSS (CDN)

## Project File

- `visit_sweden_tailwind_practice.html`

## How to Run

1. Download or clone the repository.
2. Open `visit_sweden_tailwind_practice.html` in your browser.

## Purpose

The purpose of this project is to practice Tailwind CSS by building different UI components using a Sweden travel theme.
